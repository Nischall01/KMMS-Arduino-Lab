Example use case · उदाहरण

Sweep 0° to 180° — Move a robot arm back and forth smoothly.

#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(6);
}

void loop() {
  myServo.write(0);
  delay(500);
  myServo.write(180);
  delay(500);
}
