Serial.begin()

Serial सुरु

Starts hardware Serial communication at a chosen speed (baud rate). Call once in setup() before Serial.print() or Serial.println(). On Uno, this uses pins D0 (RX) and D1 (TX) through USB.

Syntax

Serial.begin(baud);

Parameters

ParameterDescription
baudBits per second — usually 9600 (must match Serial Monitor)

Returns

Nothing (void).

When to use

  • Debug sensor values on the computer
  • See messages in Arduino IDE Serial Monitor
  • Talk to Bluetooth/GPS modules on D0/D1

Common mistakes

Baud rate Serial Monitor must use the same speed as Serial.begin(). Do not wire course components to D0/D1 while using USB Serial — see Uno Serial pins.

Example

void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.println("Ready!");
  delay(1000);
}
