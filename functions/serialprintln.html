Serial.println()

Serial प्रिन्ट

Sends text or numbers to your computer over USB and starts a new line after each message. Open Serial Monitor (Tools → Serial Monitor) to read the output.

Syntax

Serial.println(value);
Serial.print(value);   // same, but no new line

Parameters

ParameterDescription
valueText in quotes, a number, or a variable (e.g. sensor reading)

Returns

Nothing (void).

When to use

  • Print distance, temperature, or button state while testing
  • Confirm your code reached a certain line
  • Log values without an LCD screen

Common mistakes

Setup first Call Serial.begin(9600) in setup() before any print. If Serial Monitor shows garbage, check the baud rate matches.

Example

void setup() {
  Serial.begin(9600);
  pinMode(A0, INPUT);
}

void loop() {
  int light = analogRead(A0);
  Serial.print("Light: ");
  Serial.println(light);
  delay(500);
}
