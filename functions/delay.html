delay()

ढिलाइ

Stops your entire program for a set time in milliseconds — nothing else runs during the wait.

Syntax

delay(ms);

Parameters

ParameterDescription
msMilliseconds (1000 = 1 second)

Returns

Nothing (void).

When to use

  • Simple LED blink timing
  • Short pause after ultrasonic trigger
  • Waiting for tone length

Common mistakes

Blocks everything During delay(), sensors and buttons are not read. For multiple tasks use millis() instead.

Example

delay(1000);  // wait 1 second

See also

  • millis()

Used by components

  • Built-in LED
  • Ultrasonic Sensor
  • Buzzer
