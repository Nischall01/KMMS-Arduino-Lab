pinMode()

पिन मोड सेट गर्नुहोस्

Tells the Arduino how each pin will behave — like deciding whether a door is for reading signals (INPUT) or controlling devices (OUTPUT). Always run this in setup() before using a pin.

Syntax

pinMode(pin, mode);

Parameters

ParameterDescription
pinPin number, e.g. 13 or LED_BUILTIN
modeINPUT, OUTPUT, or INPUT_PULLUP

Returns

Nothing (void).

When to use

  • OUTPUT — LEDs, relay, motor direction, ultrasonic Trig
  • INPUT — IR sensor, fire sensor digital output
  • INPUT_PULLUP — button wired to GND

Common mistakes

ध्यान दिनुहोस् Forgetting pinMode before digitalWrite causes unreliable output.

Example

pinMode(13, OUTPUT);
pinMode(2, INPUT);
