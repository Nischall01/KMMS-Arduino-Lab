Servo Class

Servo.h कक्षा

Built-in Arduino library for hobby servos. Sends correct PWM pulses so you can set angles with write() instead of raw timing.

Install library · लाइब्रेरी स्थापना

Arduino IDE → Sketch → Include Library → Servo (built-in, no install needed).

Methods · विधिहरू

MethodDescription
attach(pin)Connect servo signal to pin (e.g. 6)
write(angle)Move to 0–180 degrees
read()Last written angle
detach()Release pin

Wiring to Uno · जडान

Signal → PWM pin D6, VCC → 5V, GND → GND. See Servo Motor component.

Example · उदाहरण

#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(6);
}

void loop() {
  myServo.write(0);
  delay(500);
  myServo.write(180);
  delay(500);
}
